Hexlator

A simple and light-weight hexadecimal color finder.

Download Download View Hexlator on GitHub

Why would I want to use HEXlator?

Simplicity

HEXlator is one of the easiest and quickest ways to find a hex code. It has a very clean, simple, and appealing UI. HEXlator even offers a tutorial in Main Menu > More Options.

Cross-Platform

Being written in pure Java Swing, HEXlator is Cross-Platform. This means that you can run it on Windows, OS X, Linux, Solaris, and more! All you need to run it is to have Java installed on your system.

Free

HEXlator is free to use and contribute to with a GNU license. No money, membership, or credit card number needed to download.
